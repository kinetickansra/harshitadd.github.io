Recent Projects

Visionair

Mentored by Dr.Aakanksha Chowdhery, Google Brain

A privacy preserving smartphone application that can estimate the Air Quality Index of an image that the user takes. The application uses On-Device training to ensure that no private user data (image, location) is shared with any remote cloud service. VisionAir also uses federated learning to diversify the response of the custome build global model without compromising the privacy of its client devices. Application available on the Google Play Store (Indian Subcontinent only)

Thirsty Crow

A Line Following Robot that engages a custom built electromagnetic arm (performs pick up/drop) upon identifying a specific aruco. Path planning is done using a modified version of the BFS traversal. Blender is used to render 3D models of the real-time bot and its surrounding objects on a virtual platform.
